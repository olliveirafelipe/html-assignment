HTML Tags

h1 to h6

Defines HTML headings, goes from 1 to 6 and defines the hierarchy of the titles.

p

Defines a paragraph.

a

Defines a hyperlink, usefull for conecting pages and images to a defined area.

img

Defines an image, but we have to use the css style to define the properties of the image

li

Defines lists and might come with more variations.
